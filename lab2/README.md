# Задание 1

Для варианта 7 требуется подобрать номинальные характеристики насоса $\langle q_N^0, h_N^0 \rangle$ так, чтобы расчётный режим был максимально близок к желаемой производительности $q_N^* = 76$ м³/сут и одновременно удовлетворял условию по глубине подвески насоса.

## Постановка задачи

Подбор выполняется по критерию:

$$
J = |q_N - q_N^*| + |q_N - q_N^0|
$$

Номинальные значения выбираются из дискретных рядов:

- $q_N^0 \in \{20, 30, 40, \dots, 300\}$
- $h_N^0 \in \{400, 500, 600, \dots, 3500\}$

Проверяется условие:

$$
H_N > H(4) + h^{SP}, \quad h^{SP} = 100 \text{ м}
$$

Если оно не выполняется, глубина подвески насоса увеличивается и расчёт повторяется.

## Алгоритм решения

1. Выбирается значение $q_N^0$, ближайшее к желаемой производительности $q_N^*$.
2. При $i=2$, $u=1$, $\omega=1$ оценивается желаемый напор насоса $h_N^*$.
3. Из дискретного ряда выбирается ближайшее номинальное значение $h_N^0$.
4. По уравнениям модели рассчитываются:
   - дебит насоса $q_N$,
   - напор насоса $h_N$,
   - динамический уровень $H(4)$,
   - давления $p(1)$, $p(3)$, $p(5)$, $p(4)$, $p(8)$.
5. Проверяется условие по глубине подвески.
6. При нарушении условия выполняется увеличение $H_N$ и повторный расчёт.

# Задание 2

Для нефонтанирующей скважины требуется реализовать алгоритм перебора номинальных характеристик насоса $\langle q_N^0, h_N^0 \rangle$ и выбрать оптимальную пару по минимальному значению критерия $J$.

## Постановка задачи

Ряды перебора:

- $q_N^0 \in \{20, 30, 40, \dots, 300\}$
- $h_N^0 \in \{400, 500, 600, \dots, 2000\}$

Для каждой пары рассчитывается вектор состояний:

$$
x = [q_N, h_N, H(4), p(1), p(3), p(5), p(4), p(8)]^T
$$

Затем определяется критерий:

$$
J = |q_N - q_N^*| + |q_N - q_N^0|
$$

После этого выбирается пара с минимальным значением $J$, и выполняется проверка условия:

$$
H_N > H(4) + h^{SP}, \quad h^{SP} = 100 \text{ м}
$$

При нарушении условия глубина подвески увеличивается, и расчёт повторяется.

Оптимальная пара номинальных характеристик насоса:

# Задание 3

Для варианта 7 требуется провести вычислительные эксперименты по вариации параметров модели нефонтанирующей скважины с насосом, используя параметры, найденные в задании 2.

## Базовые параметры модели

В расчётах использовались:

- $p_L = 1.5$ МПа
- $p_R' = 17.3$ МПа
- $w_R' = 10$
- $r_{GU}' = 12$
- $\beta' = 0.1707$
- $H_N = 1700$ м
- $q_N^0 = 80$ м³/сут
- $h_N^0 = 1700$ м

## Диапазоны варьирования

Исследуемые параметры:

- $\beta \in [0;1]$
- $p_R \in [0.7p_R'; 1.3p_R']$
- $w_R \in [0.6w_R'; 1.4w_R']$
- $r_{GU} \in [0.1r_{GU}'; 20r_{GU}']$
- $\omega \in [0.7; 1.3]$

## Критерий выхода на режим срыва подачи

Для каждого эксперимента рассчитывался параметр:

$$
\Delta h^{SP} = H_N - H(4) - h^{SP},
$$

где $h^{SP} = 100$ м.

Система считается вышедшей на режим срыва подачи при выполнении условия:

$$
\Delta h^{SP} = 0
$$

## Результаты задания 3

По результатам вычислительных экспериментов найдены пороговые значения параметров:

| $\beta_{SP}$ | $pR_{SP}$, МПа | $wR_{SP}$ | $\omega_{SP}$ | $rGU_{SP}$ |
|---:|---:|---:|---:|---:|
| 0.5175 | 16.3026 | 8.8320 | 1.0433 | 29.0493 |

## Анализ результатов

1. При увеличении обводнённости $\beta$ динамический уровень $H(4)$ возрастает, и при $\beta \approx 0.5175$ система достигает границы срыва подачи.

2. При уменьшении пластового давления $p_R$ производительность насоса снижается, что приводит к росту динамического уровня. Граница срыва подачи достигается при $p_R \approx 16.3026$ МПа.

3. При уменьшении коэффициента продуктивности $w_R$ система также приближается к режиму срыва подачи. Пороговое значение составило $w_R \approx 8.8320$.

4. При увеличении гидросопротивления клапана в затрубном пространстве $r_{GU}$ давление в ЗТ возрастает, что вызывает рост уровня $H(4)$. Пороговое значение составило $r_{GU} \approx 29.0493$.

5. При увеличении частоты вращения насоса $\omega$ до $\omega \approx 1.0433$ система достигает предельного состояния по условию $\Delta h^{SP} = 0$.

## Вывод по заданию 3

В ходе задания 3 установлено, что для варианта 7 наиболее критичными параметрами, влияющими на переход системы в режим срыва подачи, являются обводнённость, пластовое давление, коэффициент продуктивности, гидросопротивление клапана в затрубье и частота вращения насоса. Для каждого параметра определены пороговые значения, соответствующие условию $\Delta h^{SP} = 0$.
