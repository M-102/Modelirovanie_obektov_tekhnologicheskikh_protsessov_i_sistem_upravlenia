# Лабораторная работа 2

## Задание 1

Для варианта 7 выполнен подбор номинальных характеристик насоса по алгоритму задания 1.

### Исходные данные

- `pL = 1.5` МПа
- `γo = 0.0068` МПа/м
- `γw = 0.0109` МПа/м
- `rU = 2`
- `γG0 = 0.000008` МПа/м
- `kC = 0.7`
- `pR1 = 20.9` МПа
- `pR0 = 17.3` МПа
- `rR = 0.1` МПа·сут/м³
- `γ = 0.0075` МПа/м
- `HR = 2500` м
- `HN = 1500` м
- `rGU = 12`
- `G = 100`
- `qN* = 76` м³/сут

### Алгоритм решения

1. Выбрано номинальное значение подачи насоса `qN0`, ближайшее к желаемой производительности `qN*`.
2. По формуле модели при `i = 2` оценён желаемый номинальный напор `hN*`.
3. Из дискретного ряда типоразмеров выбран ближайший номинальный напор `hN0`.
4. По уравнениям модели рассчитаны:
   - дебит насоса `qN`
   - напор насоса `hN`
   - динамический уровень `H(4)`
   - давления `p(1)`, `p(3)`, `p(5)`, `p(4)`, `p(8)`
5. Проверено условие:
   `HN > H(4) + hSP`, где `hSP = 100 м`
6. При нарушении условия глубина подвески `HN` увеличивалась, после чего расчёт повторялся.

### Вывод

Для варианта 7 в обоих случаях оптимальным по алгоритму оказалось номинальное значение подачи насоса `qN0 = 80` м³/сут. Для фонтанирующей скважины выбран номинальный напор `hN0 = 1100`, при этом условие по глубине подвески насоса выполняется уже при `HN = 1500` м. Для нефонтанирующей скважины условие не выполнялось при исходной глубине подвески, поэтому было выполнено увеличение `HN` до `1600` м, после чего условие стало выполняться. Полученные расчётные значения дебита насоса близки к желаемой производительности `qN* = 76` м³/сут.
