# Моделирование фонтанирующей скважины — лабораторные работы

Репозиторий содержит выполнение лабораторных заданий по модели фонтанирующей скважины для варианта 7.

## Содержание

Выполнены задания:

1. Расчёт параметров скважины с выключенным насосом для двух режимов:
   - скважина фонтанирует;
   - скважина не фонтанирует.

2. Определение граничных значений параметров, при которых фонтанирование прекращается:
   - обводнённость `β`,
   - пластовое давление `pR`,
   - давление в линии `pL`.

3. Вычислительные эксперименты по вариации параметров модели:
   - `β`,
   - `pR`,
   - `pL`,
   - `wR`,
   - `rGU`.

## Структура проекта

- `notebooks/` — Jupyter/Colab ноутбуки по заданиям
- `src/` — функции модели и вычислительные модули
- `data/input/` — исходные параметры
- `data/output/` — таблицы результатов
- `reports/` — отчёт и рисунки
- `docs/` — краткое описание формул
